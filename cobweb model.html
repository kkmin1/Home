Ά│ιΉιΧΆ╕░ΈΝΑΈςρΊαΧ(static expectation model)

Ά│╡Ά╕ΚΉηΡΈΛΦ ΈΜρΉΙεΊηΙ ΉιΕΆ╕░Ήζα Ά░ΑΆ▓σΈπΝΉζΕ Ά│ιΈινΊΧαΉΩυ Ά│╡Ά╕ΚΈθΚΉζΕ Ά▓░ΉιΧΊΧεΈΜν.

ΉοΚ, $ P_t^e = P_{t-1} $

ΈπνΆ╕░(όψΠόεθ) ΉΜεΉηξΉ▓φΉΓ░Ήζ┤ Ήζ╝ΉΨ┤ΈΓαΈψΑΈκε, ΈπνΆ╕░Ήζα ΉΙαΉγΦΈΛΦ Ά│╡Ά╕ΚΆ│╝ Ά░βΈΜν.

$ a - b P_t = c + d P_t^e $ ΉΩΡΉΕε $ P_t^e = P_{t-1} $ Έξ╝ ΈΝΑΉηΖΊΧαΈσ┤

$ a - b P_t = c + d P_{t-1} $ Ήζ┤ΈψΑΈκε

$ P_t = \dfrac{a-c}{b} - \dfrac{d}{b}P_{t-1} $

Ήζ┤ΈΛΦ Ήζ╝Ή░ρ Έ╣ΕΈΠβΉ░ρ Ή░ρΈ╢ΕΈ░σΉιΧΉΜζΉζ┤ΈψΑΈκε Ά╖ιΊαΧΆ░ΑΆ▓σΉζΑ

Ή░ρΈ╢ΕΈ░σΉιΧΉΜζcobweb ΈςρΊαΧ Ή░╕Ήκ░

$ P^* = \dfrac{a-c}{b} - \dfrac{d}{b}P^* $

$ \Longrightarrow P^* = \dfrac{a-c}{b+d} $

ΈΦ░Έζ╝ΉΕε ΊΧ┤ΈΛΦ $ P_t = \dfrac{a-c}{b+d} +({- \dfrac{d}{b}}^t)(p_0 - \dfrac{a-c}{b+d}) $

$ - \dfrac{d}{b} $Ήζ┤ΈψΑΈκε $ P_t $ΈΛΦ ΉπΕΈΠβΊΧρΉζΕ ΉΧΝ ΉΙα ΉηΙΈΜν.

i) $ d > b $

Ά░ΑΆ▓σΉΩΡ ΈΝΑΊΧε Ά│╡Ά╕ΚΉζα ΊΔΕΈιξΉΕ▒Ήζ┤ ΉΙαΉγΦΉζα ΊΔΕΈιξΉΕ▒Έ│┤ΈΜν ΊΒ░ Ά▓╜Ήγ░ΉΩΡΈΛΦ Ά░ΑΆ▓σΆ│╝ Ά│╡Ά╕ΚΈθΚΉζΑ ΈΞΦΉγ▒ Ά╖╕ ΊΠφΉζΕ ΈΕΥΊηΙΈσ┤ΉΕε Ά▓╜ΈκεΈΛΦ Έ░εΉΓ░ΊΧεΈΜν.

ii) $ d = b $

Ά░ΑΆ▓σΉΩΡ ΈΝΑΊΧε Ά│╡Ά╕ΚΉζα ΊΔΕΈιξΉΕ▒Ά│╝ ΉΙαΉγΦΉζα ΊΔΕΈιξΉΕ▒Ήζ┤ Ά░βΉζΑ Ά▓╜Ήγ░ΉΩΡΈΛΦ Ά│ΕΉΗΞ Ά░βΉζΑ Ά▓╜ΈκεΈξ╝ ΉΙεΊβαΊΧεΈΜν.

iii) $ d < b $

Ά░ΑΆ▓σΉΩΡ ΈΝΑΊΧε ΉΙαΉγΦΉζα ΊΔΕΈιξΉΕ▒Ήζ┤ Ά│╡Ά╕ΚΉζα ΊΔΕΈιξΉΕ▒Έ│┤ΈΜν ΊΒ┤ Ά▓╜Ήγ░ ΉΙαΈι┤ΊΧεΈΜν. Ά╖ιΊαΧΉΩΡ Ή╢σΆ▓σΉζ┤ Ά░ΑΊΧ┤ΉπΑΈσ┤ ΉΔΙΈκεΉγ┤ Ά╖ιΊαΧΉε╝Έκε ΉΙαΈι┤ΊΧεΈΜν.

ΊαΕΉηυ Ά╖ιΊαΧΉζΑ $ P^* $, $ Q^* $ΉΩΡΉΕε ΊαΧΉΕ▒ΈΡαΉΨ┤ ΉηΙΈΜν.

ΉΨ┤ΈΨν Ήζ┤ΉειΈκε 0Ά╕░ΉΩΡ ΉιεΊΤΙΉζα Ά░ΑΆ▓σΉζ┤ Ά╖ιΊαΧΆ░ΑΆ▓σΉζ╕ $ P^* $ΉΩΡΉΕε Ήζ┤ΊΔΙΊΧαΉΩυ $ P_0 $ΉΩΡΉΕε Ά░ΑΆ▓σΉζ┤ ΊαΧΉΕ▒ΈΡΡΉζΕ Ά▓╜Ήγ░ Ά│╡Ά╕ΚΉηΡΈΛΦ 1Ά╕░ΉΩΡ $ Q_1 $ΉζΕ Ά│╡Ά╕ΚΊΧεΈΜν. ΉΜεΉηξΉΩΡΈΛΦ $ Q_1 $ ΈπΝΊΒ╝Ήζα ΉΔΒΊΤΙΉζ┤ Ήκ┤ΉηυΊΧαΆ│ι ΉΜεΉηξΆ╖ιΊαΧΉζΕ ΉεΕΊΧ┤ΉΕεΈΛΦ Ά│╡Ά╕Κ = ΉΙαΉγΦΉΩΡΉΕε $ Q_1 $Ήζα ΉΙαΉγΦΈθΚΉΩΡ ΈΝΑΊΧ┤ $ P_1 $Ήε╝Έκε Ά░ΑΆ▓σΉζ┤ Ά▓░ΉιΧΈΡεΈΜν. 1 Ά╕░ΉΩΡ $ P_1 $Ήζα Ά░ΑΆ▓σΉζΕ Έ│╕ Ά│╡Ά╕ΚΉηΡΈΛΦ 2Ά╕░ΉΩΡ $ Q_2 $Έξ╝ Ά│╡Ά╕ΚΊΧαΆ▓Ν ΈΡαΆ│ι Ήζ┤ΈΛΦ ΉΙαΉγΦΆ│╡Ά╕ΚΉζα Έ▓ΧΉ╣βΉΩΡ ΈΦ░Έζ╝ $ P_2 $Έκε Ά░ΑΆ▓σΉζ┤ Ά▓░ΉιΧΈΡεΈΜν. 2 Ά╕░ΉΩΡ $ P_2 $Έξ╝ Έ│╕ Ά│╡Ά╕ΚΉηΡΈΛΦ 3Ά╕░ΉΩΡ $ Q_3 $Ά│╡Ά╕ΚΊΧαΆ▓Ν ΈΡαΆ│ι Ήζ┤ΈΛΦ ΉΙαΉγΦΆ│╡Ά╕ΚΉζα Έ▓ΧΉ╣βΉΩΡ ΈΦ░Έζ╝ $ P_3 $Έκε Ά░ΑΆ▓σΉζ┤ Ά▓░ΉιΧΈΡεΈΜν. 3Ά╕░ΉΩΡ $ P_3 $Ήζα Ά░ΑΆ▓σΉζΕ Έ│╕ Ά│╡Ά╕ΚΉηΡΈΛΦ 4Ά╕░ΉΩΡ $ Q_4 $Έξ╝ Ά│╡Ά╕ΚΊΧαΆ▓Ν ΈΡαΆ│ι Ήζ┤ΈΛΦ ΉΙαΉγΦΆ│╡Ά╕ΚΉζα Έ▓ΧΉ╣βΉΩΡ ΈΦ░Έζ╝ $ P_4 $Έκε Ά░ΑΆ▓σΉζ┤ Ά▓░ΉιΧΈΡεΈΜν. Ήζ┤ΉβΑ Ά░βΉζΑ Ά│╝ΉιΧΉζ┤ Έ░αΈ│╡ΈΡαΉΨ┤ (Ά╖╕Έο╝ 2)ΉΩΡΉΕε Έ│┤ΈΛΦ Έ░ΦΉβΑ Ά░βΉζ┤ Ά╖ιΊαΧΆ░ΑΆ▓σΉΩΡΉΕεΉζα Ήζ┤ΊΔΙΉζΑ Ά▒░Έψ╕ΉπΣ ΈςρΉΨΣΉε╝Έκε ΈΜνΉΜε Ά╖ιΊαΧΆ░ΑΆ▓σΉε╝Έκε ΉΙαΈι┤ΊΧαΆ▓Ν ΈΡεΈΜν.

2) ΉιΒΉζΣΉιΒΆ╕░ΈΝΑΈςρΊαΧ(adaptive expectation model)

ΉιΒΉζΣΉιΒ Ά╕░ΈΝΑΊΧαΉΩΡ Ά░ΑΆ▓σ ΉαΙΉ╕κΉζΑ ΉιΕΆ╕░Ήζα ΉαΙΉ╕κΉανΉ░ρΈξ╝ Έ░αΉαΒΊΧαΉΩυ ΈΜνΉζΝΆ│╝ Ά░βΉζ┤ Ήζ┤ΈμρΉΨ┤ ΉπΕΈΜν.5

$ p_t^e - p_{t-1}^e = \lambda(p_{t-1} - p_{t-1}^e) $, $ 0 < \lambda < 1 $

$ \Rightarrow p_t^e = \lambda p_{t-1} + (1 - \lambda)p_{t-1}^e $ ..............έΣι

ΉοΚ Ά│╡Ά╕ΚΉηΡΈΛΦ ΉιΕΆ╕░Ήζα Ά░ΑΆ▓σΆ│╝ ΉαΙΉ╕κΆ░ΑΆ▓σΉζα Ά░ΑΉνΣΊΠΚΆ╖ιΉζΕ Ά│ιΈινΊΧαΉΩυ Ά│╡Ά╕ΚΈθΚΉζΕ Ά▓░ΉιΧ

ΉΜεΉηξΉ▓φΉΓ░ Ά░ΑΉιΧΉΩΡΉΕε

$ a - b P_t = c + d P_t^e $ ΉΩΡΉΕε

$ P_t = \dfrac{a-c}{b} - \dfrac{d}{b}P_t^e $

έΣιΉζΕ ΈΝΑΉηΖΊΧαΈσ┤

$ P_t = \dfrac{a-c}{b} - \dfrac{d}{b}(\lambda p_{t-1} + (1 - \lambda)p_{t-1}^e) $ ...............έΣκ

ΊΧεΊΟ╕, (t-1)Ά╕░ΈΛΦ $ P_{t-1} = \dfrac{a-c}{b} - \dfrac{d}{b}P_{t-1}^e $

ΉΨΣΈ│ΑΉΩΡ (╬╗-1)Έξ╝ Ά│▒ΊΧαΆ│ι έΣκΉΩΡ ΈΞΦΊΧαΈσ┤

$ P_t = \lambda \dfrac{a-c}{b} + (1 - \lambda - \lambda \dfrac{d}{b})p_{t-1} $

Ήζ┤ Ή░ρΈ╢ΕΈ░σΉιΧΉΜζΉζα ΊΧ┤ΈΛΦ

$ P_t = \dfrac{a-c}{b+d} + (1 - \lambda - \lambda \dfrac{d}{b})^t (p_0 - \dfrac{a-c}{b+d}) $

$ -1 < 1 - \lambda - \lambda \dfrac{d}{b} < 1 $ Ήζ┤Έσ┤ $ \lim\limits_{t \to \infty} P_t = \dfrac{a-c}{b+d} $ Ήζ┤ΈψΑΈκε ΊΧ┤ΈΛΦ ΉΧΙΉιΧΉιΒ

i) $ 1 - \lambda - \lambda \dfrac{d}{b} < 1 $ Ήζα ΉΕ▒Έο╜ΉΩυΈ╢Α

$ 1 - \lambda - \lambda \dfrac{d}{b} < 1 \quad \Rightarrow \quad \lambda + \lambda \dfrac{d}{b} > 0 $

$ \lambda, b, d > 0 $ Ήζ┤ΈψΑΈκε ΊΧφΉΔΒ ΉΕ▒Έο╜

ii) $ <-1 < 1 - \lambda - \lambda \dfrac{d}{b} $ Ήζα ΉΕ▒Έο╜ΉΩυΈ╢Α

$ <-1 < 1 - \lambda - \lambda \dfrac{d}{b} \quad \Rightarrow \quad \lambda + \lambda \dfrac{d}{b} < 2 $

$ \Rightarrow \quad \dfrac{d}{b} < \dfrac{2}{\lambda} -1 $

$ 0 < \lambda < 1 $ Ήζ┤ΈψΑΈκε $ \dfrac{2}{\lambda} - 1 > 1 $

$ \dfrac{d}{b} < 1 $Ήζ┤Έσ┤ $ \dfrac{d}{b} < 1 < \dfrac{2}{\lambda} - 1 $

ΉοΚ $ d < b $Ήζ┤Έσ┤ ΉΙαΉγΦΆ│κΉΕιΉζα ΉιΙΈΝΑΆ░ΤΉζα Ά╕░Ήγ╕Ά╕░Ά░Α Ά│╡Ά╕ΚΆ│κΉΕιΉζα ΉιΙΈΝΑΆ░ΤΉζα Ά╕░Ήγ╕Ά╕░Έ│┤ΈΜν ΊΒυΈΜνΈσ┤ ΊΧ┤ΈΛΦ Ά╖ιΊαΧΆ░ΑΆ▓σΉΩΡ ΉΙαΈι┤. Ά│ιΉιΧΆ╕░ΈΝΑΈςρΊαΧΆ│╝ Ά░βΉζΑ Ά▓░Ά│╝Ήζ┤ΈΜν.

$ \dfrac{d}{b} < 1 $Ήζ┤Έσ┤ $ \dfrac{d}{b} \lessgtr \dfrac{2}{\lambda} - 1 $

Έ╢ΑΊα╕Έξ╝ ΊΝΡΉιΧΊΧι ΉΙα ΉΩΗΈΜν. ΉοΚ ΉΙαΈι┤ΊΧι ΉΙαΈΠΕ Έ░εΉΓ░ΊΧι ΉΙαΈΠΕ ΉηΙΈΜν. ΉοΚ $ d > b $ Ήζ┤ΈΞΦΈζ╝ΈΠΕ ΉΙαΈι┤ΊΧι ΉΙαΆ░Α ΉηΙΈΜν. Ά│ιΉιΧΆ╕░ΈΝΑΈςρΊαΧΆ│╝ ΈΜνΈξ╕ Ά▓░Ά│╝Ήζ┤ΈΜν.


Ά▒░Έψ╕ΉπΣΉζ┤ΈκιΉζα ΉΓυΈκΑ

ΈΝΑΊΣεΉιΒΉζ╕ ΉαΙΈκε ΈσΦΈςρΈου Έ░αΈΠΕΉ▓┤Ήζα Ά░ΑΆ▓σΉζΕ ΈΥν ΉΙα ΉηΙΈΜν.

ΈσΦΈςρΈου Έ░αΈΠΕΉ▓┤Ήζα Ά░ΑΆ▓σΉζΑ ΉιΕΊαΧΉιΒΉζ╕ Ά▒░Έψ╕ΉπΣ ΈςρΊαΧΉζΕ ΈΦ░Έξ┤Ά│ι ΉηΙΈΜν. ΉιΕΆ╕░Ήζα Ά│╡Ά╕ΚΉζ┤ ΈΛαΉΨ┤ΈΓαΈσ┤ Ά░ΑΆ▓σΉζ┤ ΈΓ┤ΈινΆ░ΑΆ│ι Ήζ┤Έκε Ήζ╕ΊΧ┤ ΉΙαΉγΦΆ░Α ΈΛαΈσ┤ Ά│╡Ά╕ΚΉζ┤ Έ╢ΑΉκ▒ΊΧαΈψΑΈκε Ά░ΑΆ▓σΉζ┤ ΉαυΈζ╝Ά░ΑΆ│ι Ήζ┤ΈΛΦ ΉΙαΉγΦ Ά░ΡΉΗΝΈκε Ήζ┤ΉΨ┤ΉπΕΈΜν.

ΈαΡ ΈΜνΈξ╕ ΉαΙΈκε ΈΗΞΉηΣΈυ╝ΉζΕ ΈΥν ΉΙα ΉηΙΈΜν. Έ░░Ή╢Φ ΈΗΞΉΓυΆ░Α ΊΤΞΈΖΕΉζ┤ ΈΡαΉΨ┤ Ά│╡Ά╕Κ Ά│╝ΉηΚΉζ┤ ΈΡαΈσ┤ Ά╖╕ ΊΧ┤Ήζα Έ░░Ή╢ΦΈΗΞΆ░ΑΈΛΦ ΉΜ╝Ά░ΤΉΩΡ Έ░░Ή╢ΦΈξ╝ ΈπνΈπνΊΧεΈΜν. Ήζ┤Ήιε ΈΜνΉζΝ ΊΧ┤ΉΩΡΈΛΦ Έ░░Ή╢Φ Ά│╡Ά╕ΚΈθΚΉζ┤ ΊΒυΆ▓Ν ΉνΕΆ▓Ν ΈΡαΉΨ┤ Ά░ΑΆ▓σΉζ┤ Ά╕ΚΈΥ▒ΊΧαΆ▓Ν ΈΡαΆ│ι Ήζ┤ΈθυΊΧε ΊαΕΉΔΒΉζΑ Ά│ΕΉΗΞΊΧ┤ΉΕε ΉΙεΊβαΊΧαΆ▓Ν ΊΧεΈΜν. Ήζ┤ΈθυΊΧε ΉΧΖΉΙεΊβαΉζ┤ Ά│ΕΉΗΞΈΡαΆ▓Ν ΈΡεΈΜνΈσ┤ ΉΗΝΈ╣ΕΉηΡ, Ά│╡Ά╕ΚΉηΡ ΈςρΈΣΡ ΊΦ╝ΊΧ┤Έξ╝ Έ│┤Ά▓Ν ΈΡεΈΜν. Ήζ┤ΈθυΊΧε Έυ╕ΉιεΈξ╝ Ά╖╣Έ│╡ΊΧαΆ╕░ ΉεΕΊΧ┤ ΈΗΞΆ░ΑΈΜρΉεΕ ΉΗΝΈΥζΉΧΙΉιΧΉπΑΉδΡ ΉιεΈΠΕΆ░Α ΉηΙΈΜν.
