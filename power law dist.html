자기 유사성(scale free distribution)


https://exactitude.tistory.com/m/936, M.E.J. Newman, "Power Laws, Pareto Distributions and Zipf's Law" 참조.

확률분포 $p(x)$가 다음을 만족시키면 이 분포를 scale free distribution라 한다.  

$p(bx) = q(b)p(x)$, for any b, b 는 상수, x 는 변수......①

즉 x를 b배 하더라도 자기의 형태를 유지하는 분포를 말한다.

That is, if we increase the scale or units by which we measure x by a factor of b, the shape of the distribution p(x) is unchanged, except for an overall multiplicative constant.

이를 만족하는 함수는 거듭제곱분포(power law distribution)뿐이다.


증명

$p(x) = Cx^{- \alpha} $

①에 x = 1을 대입하면

$p(b) = q(b)p(1)$.........②

②를 ①에 대입하면

$ \Rightarrow g(b) = \dfrac{p(b)}{p(1)}$

$ p(bx) = \dfrac{p(b)}{p(1)}p(x)$ ....... ③

Since this equation is supposed to be true for any b, we can differentiate both sides with respect to b

③의 좌변을 미분하면 미분의 연쇄법칙에 의해

$ \dfrac{dp(bx)}{d(bx)} \dfrac{d(bx)}{db} = x \dfrac{dp(bx)}{d(bx)} $ , (by chain rule)

따라서 ③ 은 $ x \dfrac{dp(bx)}{d(bx)} = \dfrac{dp(b)/db}{p(1)}p(x)$

$ b = 1 $을 대입하면

$ x \dfrac{dp(x)}{dx} = \dfrac{dp(1)/db}{p(1)}p(x)$

$ = -\alpha p(x)$, where $\alpha = \dfrac{dp(1)/db}{p(1)} $

이제 $ xp'(x) = -\alpha p(x) $ 의 미분방정식을 푼다. 여기서 $ p'(x) = \dfrac{dp(x)}{dx} $

$ \dfrac{p'(x)}{p(x)} = - \dfrac{\alpha}{x} $

양변 적분하면

$ \ln p(x) = - \alpha \ln x + C $, C 는 적분상수 .......④

식 ④ 에 $ x = 1 $ 을 대입하면 $ \ln p(1) = C $

따라서 식 ④는 $ \ln p(x) = - \alpha \ln x + \ln p(1) $

$ \therefore p(x) = p(1)x^{- \alpha} $, where $\alpha = \dfrac{p'(1)}{p(1)} $

Thus, as advertised, the power-law distribution is the only function satisfying the scale-free criterion


멱급수 분포(power law distribution)

확률밀도함수(probability distribution function:pdf)는

$ p(x) = cx^{-\alpha} $, $ (x_{min} < x < \infty) $

모든 확률의 합은 1 이므로

$ 1 = \displaystyle \int_{x_{min}}^{\infty} p(x)dx = c \displaystyle \int_{x_{min}}^{\infty} x^{-\alpha}dx $

$ = \dfrac{c}{-\alpha+1} \biggl [x^{-\alpha+1} \biggr ]_{x_{min}}^{\infty} $

$ \alpha < 1 $ 이면 적분이 $ \infty $로 발산하므로 확률합이 1이 될 수 없다.
실제로 자연에서도 멱급수 분포의 지수가 α < 1 로 측정되는 경우는 거의 없다.

따라서 $ \alpha > 1 $에서

$ 1 = \dfrac{c}{\alpha - 1} x^{-\alpha+1}_{min} $
