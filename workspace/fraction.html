웹에서 분수, 수식 표현하는 방법

1. 그냥 역슬래쉬 /로 표기

html 코드 : 
<body>
    <p>정답은 1 1/10 입니다.</P> 
</body>

출력 : 정답은 1 1/10 입니다.

2. 역슬래쉬 대신 특수기호 &frasl; 사용

html 코드 : 
<body>
    <p>정답은 1 1&frasl;10 입니다.</P> 
</body>

출력 : 정답은 1 1⁄10 입니다.

3. 특수기호 &frasl;와 윗첨자 <sup > 아랫첨자 <sub > 사용

html 코드 : 
<body>
    <p>정답은 1 <sup > 1 </sup > &frasl; <sub >10 </sub >입니다.</P> 
</body>

출력 : 정답은  11⁄10입니다.

4. CSS 스타일 사용 $1$

html 코드 : 
<head>
.frac { display: inline-block;
    position: relative;
    vertical-align: middle;
    letter-spacing: 0.001em;
    text-align: center;
}
.frac > span {
    display: block;
    padding: 0.1em;
}
.frac span.bottom {
    border-top: thin solid black;
}
.frac span.symbol {
    display: none;
}
</head>

<body>
    정답은 1<div class="frac">
    <span>11</span>
    <span class="symbol"></span>
    <span class="bottom">2 </span></div>입니다.
</body>
출력 : 정답은  1
11 2
입니다.

5. CSS 스타일 사용 $2$

html 코드 : 
<head>
span.frac {
  display: inline-block;
  font-size: 100%;
  text-align: center;
}
span.frac > sup {
  display: block;
  border-bottom: 1px solid;
  font: inherit;
}
span.frac > span {
  display: none;
}
span.frac > sub {
  display: block;
  font: inherit;
}
</head>

<body>
    <p>출력 : 정답은 1 <span class="frac"><sup>1</sup><span>/</span><sub>10</sub></span>입니다.</p>
</body>

출력 : 정답은   1 1/10입니다.

7. xml 사용

아래 xml표현을 크롬에서 쓰기 위해서는 javascript를 설치해야 함. firefox에서는 그냥 됨.

<head>
<!-- xml 사용. -->
<script>window.MathJax = { MathML: { extensions: ["mml3.js", "content-mathml.js"]}};</script> 

</head>

<body>
<p>출력 : 정답은
<math>
  <mrow>
    <mn>1
    <mfrac>
      <mn>1
      <mn>10
    </mfrac>
  </mrow>
</math>
입니다.</p>
</body>

출력 : 정답은 1 1 10 입니다.

8. 수식 입력

x 2 4 x 4 0 a + b - c

$ x $에 관한 이차방정식 $ ax^2 + bx + c = 0 $의 해는 다음과 같다.

$$ x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} $$
